# Spring-basics(Inflearn ベース）

Spring の基本を学習し、実習コードを記録するリポジトリです。  

---

## 目的（Goals）
- ビズネス要求事項と設計
- 会員ドメイン設計
- 会員ドメイン開発
- 会員ドメイン実行とテスト
- 注文と割引ドメイン設計
- 注文と割引ドメイン開発
- 注文と割引ドメイン実行とテスト

## 環境（Environment）
- **Build**: Gradle（**Groovy DSL**）
- **Spring Boot**: 4.0.0
- **Java**: 21
- **IDE**: IntelliJ IDEA
- **Dependencies**: Spring Web, Thymeleaf
